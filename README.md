# 📝 Python To-Do List

A simple command-line To-Do List application built using Python.

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* 🗑️ Clear all tasks
* 🚪 Exit the application

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project in VS Code.
3. Run the following command:

```bash
python todo_list.py
```

## 📷 Sample Menu

```
===== TO-DO LIST =====

1. Add Task
2. View Tasks
3. Delete Task
4. Clear All Tasks
5. Exit
```
